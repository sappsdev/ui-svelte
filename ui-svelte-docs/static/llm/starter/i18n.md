## Internationalization (i18n)

Built-in system for multi-language support with translations, date/number formatting, and pluralization.

### Setup

**1. Create translation files in `src/lib/locales/`:**

```typescript
// src/lib/locales/en.ts
export default {
  common: {
    welcome: "Welcome, {{name}}!"
  },
  messages: {
    items: {
      singular: "You have {{count}} item",
      plural: "You have {{count}} items"
    }
  }
};

// src/lib/locales/es.ts
export default {
  common: {
    welcome: "Bienvenido, {{name}}!"
  },
  messages: {
    items: {
      singular: "Tienes {{count}} artículo",
      plural: "Tienes {{count}} artículos"
    }
  }
};
```

**2. Initialize in `src/hooks.client.ts`:**

```typescript
import { initLanguage } from "ui-svelte";

initLanguage({
  defaultLanguage: "en",
  availableLanguages: ["en", "es"]
});
```

### Available Functions

| Function | Description | Example |
|----------|-------------|---------|
| `t(key, params?)` | Translate with parameters | `t("common.welcome", { name: "John" })` |
| `plural(key, count)` | Handle pluralization | `plural("messages.items", 5)` |
| `formatDate(date, options?)` | Locale-aware date formatting | `formatDate(new Date(), { dateStyle: "medium" })` |
| `formatNumber(number)` | Locale-aware number formatting | `formatNumber(1234567.89)` |
| `formatCurrency(amount, currency)` | Currency formatting | `formatCurrency(99.99, "EUR")` |
| `i18n()` | Access i18n store | `const store = i18n(); store.setLanguage("es")` |

### Usage Example

```svelte
<script lang="ts">
  import { t, plural, formatDate, formatCurrency, formatNumber, i18n } from "ui-svelte";
  
  const store = i18n();
  let itemCount = $state(5);
</script>

<h1>{t("common.welcome", { name: "Alice" })}</h1>
<p>{plural("messages.items", itemCount)}</p>
<p>{formatDate(new Date(), { dateStyle: "full" })}</p>
<p>{formatNumber(1234567.89)}</p>
<p>{formatCurrency(99.99, "USD")}</p>

<button onclick={() => store.setLanguage("es")}>Switch to Spanish</button>
```

### Configuration Options

| Option | Type | Description |
|--------|------|-------------|
| `defaultLanguage` | `string` | Default language code (e.g., "en") |
| `availableLanguages` | `string[]` | Supported language codes (e.g., ["en", "es"]) |

### Translation File Structure

- **File naming**: Must match language code exactly (`en.ts`, `es.ts`, `fr.ts`)
- **Parameters**: Use `{{param}}` for interpolation
- **Pluralization**: Use `singular` and `plural` keys with `{{count}}`

**For LLMs**: Import functions from `'ui-svelte'`. Initialize i18n in `hooks.client.ts`. Use `t()` for translations, `plural()` for count-based, and `format*()` for locale-aware formatting. Language changes are reactive - no page reload needed.
